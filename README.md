# Bolt Automation Framework

Automation framework built using:

- Java 17
- Selenium WebDriver
- Cucumber BDD
- Rest Assured
- Spring Boot
- Maven
- Jenkins CI/CD
- Allure Reports
- GitHub

## Features
- UI Automation
- API Automation
- Parallel Execution
- Thread-safe WebDriver
- Logging
- Screenshot Capture
- Jenkins Pipeline
- Allure Reporting
